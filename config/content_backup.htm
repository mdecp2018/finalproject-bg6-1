git基本指令

  

cd name   進入資料夾

git init   初始一個可管理的倉儲

git add name  將資料先暫存到 staging area

git commit -m ""   將staged檔案提交到儲存庫中

git status  查看目前git倉庫狀態

git log  最近到最遠的提交狀況

git push 上傳資料

mkdir name 創建一個文檔夾

git checkout --file  還原誤刪的git文檔

 git branch name 創建分支

 git checkout name 切換分支

 git branch 查看目前分支
